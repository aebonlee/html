### 1. 맞춤 과정 카드 및 상세 페이지 추가

**커밋**: `f487113`

#### 변경 내용
교육신청(`#page=edu`) 페이지에 기존 3개 과정(입문/중급/고급) 외에 4번째 **맞춤 과정** 카드 추가.

#### 과정 개요
| 항목 | 내용 |
|------|------|
| 배지 | 맞춤 (Custom) |
| 제목 | 맞춤 과정 (포트폴리오 & 웹사이트 제작) |
| 레벨 색상 | `#EA580C` (Orange) |
| 교육 기간 | 4~8주 (협의 후 결정, 주 1~3회) |
| 대상 | 취준생·프리랜서, 회사 대표·실무자, 특정 프로젝트 목표자 |
| 선수 조건 | 없음 (수강생 수준에 맞춰 커리큘럼 구성) |
| 수료 역량 | 실제 운영 가능한 웹사이트 완성 및 독립 유지·관리 |

#### 커리큘럼 (5단계)
| 단계 | 제목 | 주요 내용 |
|------|------|----------|
| 사전 미팅 | 목표 설정 및 기획 | 요구사항 분석, 레퍼런스 조사, 사이트맵·와이어프레임, 맞춤 커리큘럼 수립 |
| 1단계 | 디자인 & 퍼블리싱 | 브랜드 아이덴티티, AI 디자인 시안, HTML/CSS 퍼블리싱, 반응형 레이아웃 |
| 2단계 | 기능 구현 | JS 인터랙션, 문의 폼·지도 연동, UI 컴포넌트, AI 코드 생성 |
| 3단계 | 콘텐츠 & 최적화 | 실제 콘텐츠 적용, SEO 설정, 성능 최적화, 크로스 브라우저 테스트 |
| 4단계 | 배포 & 운영 | 도메인·호스팅, SSL, Google Analytics, 유지보수 안내 |

#### 카드 특징
- 기존 `.edu-card` 스타일 재사용 (추가 CSS 불필요)
- features 4개: 개인 포트폴리오, 회사/브랜드 홈페이지, 쇼핑몰·랜딩 페이지, 1:1 맞춤 커리큘럼
- 한/영 이중 언어 완전 지원

#### 관련 변경
| 항목 | Before | After |
|------|--------|-------|
| 과정 수 | 3개 (입문/중급/고급) | 4개 (+맞춤) |
| `selectedEduCourse` 범위 | `0~2` | `0~3` |
| `levelColors` 배열 | 3개 (녹/청/보) | 4개 (+오렌지) |

| 카드 그리드 minmax | `280px` | `260px` (4카드 배치 최적화) |

#### URL 딥링크
